FAQ

Apa saja misi yang tersedia di aplikasi ini?

Aplikasi ini menawarkan lima misi harian: makan, menyikat gigi, waktu tidur, merapikan, dan persiapan sekolah.

Apakah aplikasi ini aman untuk anak-anak?

Ya, Lumi Mission Planet Pro dirancang khusus untuk anak-anak dan bebas dari iklan.

Bagaimana cara membuat misi kustom?

Anda dapat membuat misi kustom dengan memilih ikon dan latar belakang sesuai keinginan Anda.

Apakah aplikasi ini tersedia di perangkat iPhone?

Ya, Lumi Mission Planet Pro tersedia untuk diunduh di App Store untuk perangkat iPhone.

Apakah ada biaya langganan untuk menggunakan aplikasi ini?

Tidak, aplikasi ini hanya memerlukan satu kali pembelian untuk akses penuh tanpa langganan.
