ScanTo Pro: 오프라인 PDF이란?

ScanTo Pro: 오프라인 PDF은(는) iOS 앱입니다. 오프라인 PDF 스캐너·OCR

'문서 속 글자를 읽는 것'까지 기기 안에서 끝내고, 절대 온라인이 되지 않는 스캐너.

ScanTo Pro는 계약서, 영수증, 양식, 신분증, 메모를 몇 초 만에 선명한 PDF로 바꿔 줍니다. 그리고 대부분의 스캐너가 구독이나 클라우드 없이는 못 하는 일을 합니다 — 당신의 iPhone에서 문서 안의 글자를 읽어, 나중에 검색할 수 있게 합니다.

문서 '안의 글자'를 검색 — 그것도 오프라인으로

• 기기 내 문자 인식(OCR)이 모든 스캔을 자동으로 읽습니다.

• 단어를 검색하면 그것이 들어 있는 문서를 찾습니다(파일 이름만이 아닙니다).

• 스캔은 영수증·신분증·명함·계약서 등으로 자동 분류됩니다 — 모두 오프라인.

• 인식된 텍스트는 보고 복사하거나 .txt로 내보낼 수 있으며, 기기를 떠나지 않습니다.

잠금과 암호화

• Face ID 또는 암호로 앱을 잠그세요.

• 암호로 보호되는 암호화 PDF를 내보내 민감한 문서를 안전하게 공유하세요.

프라이버시 중심 설계

• 클라우드 없음: 스캔이 서버에 닿지 않습니다.

• 계정 불필요: 가입도, 로그인도, 프로필도 없습니다.

• 추적 없음: 분석·광고·서드파티 SDK 전혀 없음.

• 통신 없음: 비행기 모드에서도 똑같이 작동합니다.

무료로 시작, 한 번에 잠금 해제

무료로 다운로드하고 첫 스캔은 무료로 체험하세요. 한 번의 구매로 무제한 스캔과 워터마크 영구 제거는 물론 OCR 텍스트 내보내기, 암호화 PDF, 신분증 레이아웃 등 모든 고급 내보내기를 사용할 수 있습니다. 구독 없음, 광고 없음, 슬그머니 결제되는 무료 체험 없음.

필요한 모든 것

• 자동 가장자리 인식·원근 보정의 빠른 여러 페이지 스캔.

• 5가지 스타일: 원본·컬러·그레이스케일·흑백·문서 보정.

• 언제든 페이지 정렬·회전·삭제·추가.

• 내용 기반 스마트 자동 이름 지정.

• 신분증 모드: 앞뒤를 한 페이지에.

• 이미지에 맞춤·A4·US 레터로 PDF 내보내기 — 바로 인쇄 가능.

• 파일에 저장하거나 메일·AirDrop·메시지로 공유.

• Siri·Spotlight·단축어·액션 버튼으로 핸즈프리 스캔.

• 디테일까지 설계된 프리미엄 인터페이스.

• 33개 표시 언어 지원 (자동 감지 또는 수동 전환).

당신의 문서는 이 기기 안에만 있습니다. 그것이 이 앱의 전부입니다.
