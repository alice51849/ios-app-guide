Snapport: 여권·신분증 사진이란?

Snapport: 여권·신분증 사진은(는) iOS 앱입니다. 크기 맞춤·기기 내 제작

여권, 비자, 신분증 사진을 집에서 차분하고 안전하게 만드세요. Snapport는 일반적인 크기 템플릿, 얼굴 정렬 가이드, 배경 도구, 인쇄용 레이아웃을 제공하며 모든 처리는 iPhone에서 바로 이루어집니다.

한 번 구매. 구독 없음.

Snapport는 한 번 구매하면 계속 사용할 수 있습니다. 구독, 앱 내 구입, 광고, 추적, 계정이 없습니다.

크기 선택

• 여러 지역의 여권, 비자, 신분증, 학생증, 이력서, 사원증 사진을 위한 일반 템플릿

• 미국 2×2, 영국/솅겐/일본/한국/호주 35×45mm, 캐나다 50×70mm, 중국 33×48mm, 인도 2×2 등 대표 크기 포함

• 사용자 지정 크기 생성 가능(mm / inch / px, 원하는 DPI, 원하는 배경)

• 템플릿은 2026년 기준을 바탕으로 하며, 사용자가 원할 때 직접 실행하는 오프라인 친화적 “템플릿 업데이트 확인”을 제공합니다. 공개 JSON 일반 크기 목록만 다운로드하며 개인 데이터는 보내지 않습니다

안심하고 정렬

• Apple Vision 얼굴 감지를 기기에서 수행하고 머리 위, 눈높이, 턱 가이드를 표시

• 핀치, 드래그, 회전 지원. 얼굴 자동 중앙 정렬과 언제든지 초기화

배경 및 보정

• 배경을 흰색, 회색, 파란색, 빨간색으로 바꾸거나 원본 유지

• 기기 내 인물 분할로 전후 결과 확인

• 신원 확인에 무리가 없는 가벼운 조정(밝기, 대비, 색온도, 노출, 선명도)

내보내기 및 인쇄

• JPEG, PNG 또는 PDF로 내보내기

• 4×6, A4, Letter 인쇄 시트 생성. 여러 장 배치와 재단 가이드 지원

• 사진 또는 파일에 저장하거나 공유

프라이버시 중심 설계

• 모든 처리는 기기에서 이루어지며 사진은 업로드되지 않습니다

• 계정 없음, 광고 없음, 추적 없음, 클라우드 없음

• 완전 오프라인으로 작동합니다. 유일한 네트워크 사용은 사용자가 선택해 직접 시작하는 템플릿 업데이트 확인입니다

중요

사진 요구 사항은 국가, 기관, 신청 유형에 따라 다릅니다. Snapport는 일반 템플릿과 가이드를 제공하지만 승인 보장을 하지 않으며 어떤 정부 기관과도 관련이 없습니다. 제출 전에 반드시 공식 요구 사항을 확인하세요.
